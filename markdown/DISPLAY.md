    
## Display Element

The `display` property is used to set the display of an element.  

```css
body {
  display: block;
}
```  

```html
<style>
  .my-div-display {
    border: 10px solid red;
    display: inline;
  }
</style>

<p>This is a paragraph <div class="my-div-display">with some text</div> in it.</p>
```
