# Links
When it comes to links `<a>` there are different 4 main states you can style.  

```css
a:link {
  color: red;
}

a:visited {
  color: green;
}

a:hover {
  color: hotpink;
}

a:active {
  color: blue;
}
```

### Links Exercise
Create a page with some links. Try to be creative and use different states to style the links.
